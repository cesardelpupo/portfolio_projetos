# Portfólio de Projetos de Dados – Cesar Del Pupo

Este repositório possui o código-fonte e apresenta meu **portfólio de projetos de engenharia e análise de dados**, com foco em resolver problemas reais de negócio.  

O site reúne iniciativas que demonstram meu trabalho com **pipelines de dados, modelagem dimensional, visualização e análise estratégica**, conectando contexto de negócio e dados para gerar impacto mensurável.  

## Tecnologias utilizadas

![Python](https://img.shields.io/badge/python-3670A0?style=flat-square&logo=python&logoColor=ffdd54) ![SQL](https://img.shields.io/badge/sql-4479A1?style=flat-square&logo=postgresql&logoColor=white) ![Power Bi](https://img.shields.io/badge/power_bi-F2C811?style=flat-square&logo=microsoftpowerbi&logoColor=black) ![Git](https://img.shields.io/badge/git-%23F05033.svg?style=flat-square&logo=git&logoColor=white)

- **Engenharia:** Python, SQL, Excel
- **Controle de versão:** Git e GitHub  
- **Visualização:** Power BI, Plotly, Looker
- **Frontend:** HTML5, CSS3, JavaScript  

## Projetos em destaque

### [📊 Patient Adherence Data Pipeline](https://github.com/cesardelpupo/patient-adherence-data-pipeline)
*Built an **end-to-end data pipeline** to analyze patient attendance behavior and its impact on revenue predictability, transforming manual records into a structured analytical model using **Python**, **SQL**, and **Power BI**.*

### [🏠 Airbnb NYC Data Analysis](https://github.com/cesardelpupo/airbnb_nyc)
*Developed a data analysis project using the Airbnb NYC dataset, performing ETL and dimensional modeling in **Power Query**, and delivering insights on supply concentration, pricing distribution, and user engagement through **Power BI**.*

## Como visualizar o portfólio

Você pode visualizar o site diretamente via GitHub Pages: 

Acesse o portfólio aqui: [Portfólio de Projetos - Cesar Del Pupo](https://cesardelpupo.github.io/portfolio_projetos/)

Ou clonando este repositório e abrindo o arquivo `index.html` em seu navegador:

```bash
git clone https://github.com/cesardelpupo/portfolio_projetos.git
cd portfolio_projetos
```
